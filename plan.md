# Plan - Application de Gestion de Budget Personnel

## Vue d'ensemble
Application web complète de gestion de budget personnel avec tableau de bord, suivi des revenus/dépenses, catégorisation, budgets, objectifs d'épargne et rapports.

## Stage 1 - Design & Architecture
- Charger `vibecoding-webapp-swarm` skill
- Définir l'architecture des données (transactions, catégories, budgets, objectifs)
- Concevoir l'UI/UX avec palette sobre et professionnelle
- Créer les maquettes et le design system

## Stage 2 - Développement
- Initialiser le projet React + TypeScript + Tailwind + shadcn/ui
- Développer les composants : Layout, Sidebar, Dashboard, Transactions, Budgets, Goals, Reports
- Implémenter les visualisations (Recharts)
- Gérer l'état avec React Context / Hooks
- Persistance des données (localStorage)
- Logique métier (calculs, filtres, agrégations)

## Stage 3 - Validation & Déploiement
- Tests et validation fonctionnelle
- Déploiement de l'application

## Stack Technique
- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Recharts (graphiques)
- Lucide React (icônes)
- Framer Motion (animations)
